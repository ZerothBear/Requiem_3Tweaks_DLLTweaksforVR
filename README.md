SkyrimVR port of project 3Tweaks' patch titled "DLLTweaks" from their mod "Requiem - 3Tweaks (3BFTweaks)" (ref. https://www.nexusmods.com/skyrimspecialedition/mods/44044). 

Unfortunately, this DLL is not compatible with SkyrimVR and immediately crashes on game loading. Since it is included in LoreRim, which I am making compatible with SkyrimVR, I have re-created it for SkyrimVR and SKSEVR.

For added flexibility, I have also included an MCM menu which allows toggling the 5 fixes/tweaks on and off individually, as well as adjust the values for the Spell Absorption Chance cap (set at 75% in the original mod) and the Jump Height modifier while sneaking (set at 0.40 in the original mod).
