SkyrimVR port of project 3Tweaks' patch titled "DLLTweaks" from their mod "Requiem - 3Tweaks (3BFTweaks)" (ref. https://www.nexusmods.com/skyrimspecialedition/mods/44044). 

Unfortunately, this DLL is not compatible with SkyrimVR. Since it is included in LoreRim, which I am making compatible with SkyrimVR, I have re-created it for SkyrimVR and SKSEVR.
